# Go data access sample

it impliments go tutorial sample with postgreSQL.

## Preparetion
1. install postgreSQL
2. create database and table according to tutorial
   https://go.dev/doc/tutorial/database-access

## Usage
1. git clone
2. run
   ```
   $ go run .
   ```
